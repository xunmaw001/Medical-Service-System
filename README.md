# Medical-Service-System
该医疗服务系统使用springboot+vue开发。后端springboot能确保系统的高效稳定，方便进行数据存储、业务逻辑处理等操作。前端vue构建出简洁易用的界面，提升用户体验。系统涵盖预约挂号、病历管理、在线咨询、药品管理、医疗报告查询等功能，为患者、医生和医疗机构提供便捷服务，实现线上医疗流程的信息化，优化医疗资源配置，提升医疗服务效率。
